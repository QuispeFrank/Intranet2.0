
## Installation

Install my-project with virtualenv

```bash
  1. pip install virtualenv
  2. virtualenv venv --python=python3.8.10
  #windows:
  3. venv\scripts\activate
  #linux:
  3. source venv/bin/activate

  4. pip install -r requirements.txt
```
